Ti trovi in un progetto Laravel 12 con PHP 8.4+.
Framework UI: Filament 4.x (usa i componenti Blade e le classi Livewire corrette).
Database: MySQL 8.x.
Pacchetti chiave:

- Spatie Media Library (usa le nuove sintassi per le conversioni).
- Laravel Socialite (per l'autenticazione OAuth).
- Spatie Permissions (se presente).

Segui rigorosamente le convenzioni di Laravel:

- Usa le "Constructor Property Promotion".
- Preferisci le "Anonymous Migrations".
- Per Filament, usa sempre i metodi statici `form()` e `table()` correttamente tipizzati.
